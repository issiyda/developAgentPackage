# 開発プロジェクト知識ベース

このファイルは開発中に遭遇した問題と解決策を自動的に記録します。

## 目的
- 実装で詰まった時の解決策を蓄積
- 同じ問題の再発を防ぐ
- チーム内での知見共有
- プロジェクト固有のベストプラクティスを文書化

## 使い方

このファイルは`.claude/hooks/capture-development-knowledge.sh`によって自動的に更新されます。

問題解決や修正に関するリクエストをClaude Codeに送信すると、自動的にここに記録されます。

### 記録されるトリガーワード
- エラー、バグ、修正、問題
- うまくいかない、動かない、失敗
- 詰まった、解決、対処
- fix, error, bug, issue, problem
